# CarbonCode Privacy Policy

**Last Updated:** January 2026

## Overview

CarbonCode is a free, offline C++ IDE. We respect your privacy and are committed to being transparent about any data we collect.

## What Data We Collect

If you choose to opt in to anonymous usage analytics, we collect the following **anonymous event counters only**:

| Event | Description |
|-------|-------------|
| `app_launch` | When the IDE starts |
| `file_created` | When you create a new file |
| `file_opened` | When you open a file |
| `code_compiled` | When you compile code |
| `code_run` | When you run code |
| `debug_started` | When you start a debugging session |

## What We Do NOT Collect

- ❌ Your name, email, or any personal information
- ❌ Your code or file contents
- ❌ File names or paths
- ❌ IP addresses (anonymized by Google Analytics)
- ❌ Any information that could identify you

## Why We Collect This Data

- To understand which features are most used
- To document the IDE's adoption for academic/portfolio purposes
- To prioritize improvements and new features

## Your Control

- **First Launch**: You will be asked if you want to share anonymous usage data
- **Settings**: You can change your preference at any time in Settings → "Share anonymous usage data"
- **Offline by Default**: If you decline, no data is ever sent

## Data Processing

Anonymous event data is sent to Google Analytics 4. Google's privacy policy applies to the processing of this data: https://policies.google.com/privacy

## Contact

For privacy questions, please open an issue on our GitHub repository.

---

**Summary**: CarbonCode only collects anonymous feature usage counts, only if you opt in, and you can disable it anytime in Settings.
